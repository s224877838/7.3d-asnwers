# Stephanie's Animal Rescue 🐾

An educational mini-game for kids to learn about environmental conservation while having fun rescuing animals.

 The features include:

- The avatar selection (Fox, Otter, Owl) cna select whichever they want.
  
- Interactive mini-game: Clean trash to rescue otters help do something good in the scenario.
  
- Progress bar and educational tip to learn the facts.
  
- Reward screen after mission completion for the feeling of accomplishment.
  
- Accessibility settings for reeucing sound.
